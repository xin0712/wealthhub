## 实践指南（时间轴）

<div className="practice-timeline">
<details open>
  <summary><b>按时间推进的执行路径</b></summary>

  ### 今天（第1天）：成长股投资觉醒

  1. 完成投资理念自我诊断
  - 诚实评估你当前的投资方式：是否过度关注短期价格波动？
  - 回顾你的投资决策过程：主要基于技术分析还是基本面分析？
  - 检视你的持股时间：平均持有期是多长？是否频繁交易？
  - 成功标准：完成一份诚实的投资现状分析，识别出当前投资方式的问题和改进空间。

  2. 建立“企业家思维”
  - 将 Fisher 的核心原则写在显眼位置：“买股票就是买企业”
  - 制定企业分析检查清单：
  - 这家企业的商业模式是什么？
  - 它的竞争优势在哪里？
  - 管理层是否值得信赖？
  - 它有多大的成长空间？
  - 成功标准：开始用企业家的眼光看待股票投资，关注企业的长期价值而非短期价格。

  3. 学习 Fisher 十五要点框架
  - 研读十五要点评估体系
  - 选择一家熟悉公司，尝试用十五要点进行初评
  - 识别评估过程中的困难与知识盲区
  - 成功标准：熟悉十五要点并能初步应用。

  ### 7天内：建立成长股投资基础框架

  1. 深入学习企业分析方法（第2–3天）
  - 阅读并分析企业年报：
  - 业务描述与发展战略
  - 管理层致股东信
  - 财务数据趋势分析
  - 风险因素披露
  - 学习行业分析方法，梳理行业趋势与竞争格局
  - 建立企业分析模板，嵌入十五要点具体评估标准
  - 成功标准：能系统化分析企业基本面并形成流程。

  2. 掌握“闲聊调研法”（第4–5天）
  - 多渠道收集信息：
  - 客户口碑与产品评价
  - 供应商/合作伙伴反馈
  - 行业专家与分析师观点
  - 员工与前员工评价
  - 练习信息交叉验证，建立信息收集与整理系统
  - 成功标准：掌握一手信息的获取与验证。

  3. 建立投资研究系统（第6–7天）
  - 设立研究文件夹，分类整理资料
  - 建立企业跟踪清单：
  - 重点企业列表与基本面评分
  - 关键财务与业务指标
  - 重要事件与新闻跟踪
  - 制定定期复盘与更新计划
  - 成功标准：形成系统化的研究与跟踪体系。

  4. 开始实践企业分析（第7天）
  - 选择 2–3 家不同行业优质企业做深度分析
  - 应用十五要点进行全面评估
  - 产出报告：商业模式/竞争优势/管理层/成长前景
  - 成功标准：完成至少 2 家企业的深度分析报告。

  ### 30天内：启动成长股投资实践

  1. 构建投资候选池（第8–15天）
  - 筛选符合 Fisher 标准的成长股：行业长期增长、可持续护城河、管理层诚信能干、财务健康
  - 对候选企业做十五要点评估，建立评分与优先级
  - 成功标准：形成 10–15 家优质企业候选池。

  2. 深化调研能力（第16–20天）
  - 对重点企业开展深调：
  - 参加业绩会/电话会
  - 阅读卖方报告与专家分析
  - 关注企业官微/IR 发布
  - 客户与合作伙伴回访
  - 建立调研数据库并多角度验证
  - 成功标准：对重点企业形成一手、可信的认知。

  3. 制定投资策略（第21–25天）
  - 设定集中度（建议 5–10 只）与买卖标准
  - 风控：预案、仓位与再平衡规则
  - 时间框架：建议 3–5 年以上
  - 资金分配与决策流程固化
  - 成功标准：形成可执行的成长股投资策略。

  4. 小额实践（第26–30天）
  - 选择 1–2 只最符合标准的公司小额建仓
  - 记录：买入理由与预期、买入时企业基本面、后续跟踪与调整
  - 定期监控组合
  - 成功标准：完成首次实践并建立跟踪系统。

  ### 60–90天：完善成长股投资体系

  1. 优化投资组合（第31–45天）
  - 增加符合标准的优质企业，剔除不达标标的
  - 分散于不同子行业，避免单一风险
  - 建立组合评估与再平衡机制
  - 成功标准：形成 5–8 只优质成长股的集中组合。

  2. 深化企业跟踪（第46–60天）
  - 定期复盘财报与经营数据
  - 跟踪行业格局与管理层变动
  - 评估是否持续满足十五要点；建立预警机制
  - 成功标准：完善的跟踪与风险预警体系。

  3. 培养长期心态（第61–75天）
  - 关注企业长期价值，不被短期波动干扰
  - 恐慌期做增配预案，做好资金管理
  - 建立投资日记，记录认知与决策
  - 成功标准：具备长期主义与逆向思维的执行力。

  4. 建立个人投资哲学（第76–90天）
  - 总结 90 天学习与实操
  - 形成成长股投资原则与方法论
  - 制定 3–5 年目标与持续学习计划
  - 成功标准：沉淀可复用的投资哲学体系。
