## 实践指南（时间轴）

<div className="practice-timeline">
<details>
  <summary><b>按时间推进的执行路径</b></summary>

  ### 今天
  - [ ] 注册多个证券账户：选择至少 2–3 家提供港股打新的券商，完成开户并进行身份验证。
  - [ ] 了解港股打新基本知识：阅读关于港股打新的基础资料，确保理解打新流程和相关术语。
  - [ ] 设定资金预算：决定每次打新的投资金额，建议每次不超过总资金的 10%。

  ### 7天内
  - [ ] 学习招股说明书的阅读技巧：找到至少一份招股说明书，练习快速阅读其重要部分（如重要提示、概要、风险因素）。
  - [ ] 关注即将上市的新股：使用券商提供的工具，列出未来一周内即将上市的新股及其基本信息。
  - [ ] 模拟申购：在券商平台上进行一次模拟申购，熟悉操作流程和所需信息。

  ### 30天内
  - [ ] 参与至少 2–3 次港股打新：根据市场情况和招股说明书选择新股进行申购，记录每次的申购结果。
  - [ ] 分析新股表现：对已申购的新股进行跟踪，记录其上市后的表现（如开盘价、收盘价、涨跌幅）。
  - [ ] 学习超额认购倍数的影响：研究已上市新股的超额认购倍数，分析其对股价表现的影响。

  ### 60–90天
  - [ ] 评估投资策略：回顾过去两个月的打新经历，分析哪些策略有效，哪些需要改进，制定新的投资计划。
  - [ ] 深入研究特定行业的招股书：选择一个感兴趣的行业，分析该行业内的多家公司招股书，了解行业趋势和竞争格局。
  - [ ] 建立风险管理机制：制定一套风险管理规则，包括止损策略、资金分配和心理预期，以应对可能的破发风险。

</details>
