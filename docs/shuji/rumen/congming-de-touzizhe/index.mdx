## 实践指南（时间轴）

<div className="practice-timeline">
<details>
  <summary><b>按时间推进的执行路径</b></summary>

  ### 今天（第1天）：价值投资理念觉醒

  1. 完成投资者类型自我评估（1小时）
  - 判断自己是防御型还是积极型投资者
  - 防御型：安全、简单；积极型：愿深入研究
  - 大多数人应选择防御型策略

  2. 建立“安全边际”思维模式（30分钟）
  - 仅在价格远低于内在价值时买入
  - 原则：无足够安全边际，绝不投资
  - 座右铭：“价格是你付出的，价值是你得到的”

  3. 与“市场先生”建立正确关系（30分钟）
  - 认识市场情绪化，学会利用而非跟随
  - 不因下跌恐慌卖出，不因上涨盲目追高
  - 把波动当机会，而不是威胁

  ### 7天内：建立价值投资基础框架

  1. 学习基本财务分析技能
  - 读懂资产负债表、利润表、现金流量表
  - 计算与理解 PE、PB、流动比率、债务权益比等指标

  2. 制定股票筛选标准
  - 依据格雷厄姆七大选股标准形成个人清单
  - 防御型严格执行；积极型可适度调整
  - 建立股票观察清单并开始跟踪

  3. 确定资产配置策略
  - 股票/债券 50:50 起步；年轻可至 75% 股票，年长可降至 25% 股票
  - 偏离目标 10% 触发再平衡

  4. 选择合适的投资工具
  - 防御型：低成本宽基指数基金（如标普500）
  - 积极型：准备进行个股研究并选择合适券商

  ### 30天内：启动价值投资实践

  1. 建立核心投资组合
  - 防御型：按配置买入指数与债券基金（尽量选择费率 &lt;0.2%）
  - 积极型：建立个股组合，首批投入 ≤25% 总资金，保留弹药

  2. 实施定期投资计划
  - 每月定投 10–20% 月收入，固定日期，不择时
  - 使用“美元成本平均法”培养纪律

  3. 深入分析 3–5 只目标股票（积极型）
  - 研究商业模式、竞争地位、财务状况、管理层质量
  - 估算内在价值并给出合理买入区间

  4. 建立投资记录系统
  - 记录每笔投资的时间、价格、数量和理由
  - 建立投资日记，记录市场波动时的心理状态和决策过程

  ### 60-90天：完善价值投资体系

  1. 完成目标投资组合构建
  - 达到预设的资产配置比例，确保分散
  - 防御型至少持有 10 只不同行业股票或相应指数基金
  - 积极型个股控制在 10–30 只，避免过度分散

  2. 建立估值与监控体系
  - 为每只持仓建立估值模型，定期更新内在价值
  - 卖出触发：股价≥内在价值 90% 考虑卖出；基本面恶化立即卖出
  - 建立季度组合回顾机制

  3. 培养逆向投资心理
  - 下跌中保持冷静，把下跌视为低价买入机会
  - 识别情绪极端：他人恐惧时贪婪，他人贪婪时谨慎
  - 建立市场极端情形的心理与资金预案

  4. 制定长期投资规划
  - 设定 5/10/20 年目标；牛市谨慎、熊市积极
  - 建立投资纪律检查清单，定期自我评估是否偏离价值投资原则
