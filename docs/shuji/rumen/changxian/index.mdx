## 实践指南（时间轴）

<div className="practice-timeline">
<details open>
  <summary><b>按时间推进的执行路径</b></summary>

  ### 今天（第1天）：股票投资理念觉醒

  1. 完成股票投资现状全面审视（2小时）
  - 查看你目前的投资组合构成，记录股票、债券、现金等各类资产的具体比例
  - 计算股票投资占总资产的百分比，评估是否与你的年龄与风险承受力匹配
  - 若还未开始股票投资，从今天起建立账户与目标即为觉醒的开始

  2. 学习西格尔核心投资理念（1小时）
  - “时间是股票投资者最好的朋友”，长期回报来自生产率与股息再投资
  - 历史数据：220 年真实年化回报 股票≈6.9%、债券≈3.6%、黄金≈0.6%
  - 关键事实：持有期≥20 年，股票真实回报未出现过为负的情形

  3. 制定长期投资承诺（30分钟）
  - 写下投资目标和时间框架，承诺 ≥10 年投资期限
  - 接受短期波动是获得长期回报的代价，不因短期市场波动改变策略

  ### 7天内：建立股票投资基础认知

  1. 深入研究指数基金投资策略
  - 了解为何指数基金是普通投资者的最佳选择
  - 研究宽基指数（标普500、全市场等），对比费用率、跟踪误差、历史表现
  - 选择 3–5 只优质股票指数基金作为核心标的

  2. 学习股票估值基础知识
  - 掌握 P/E 与 CAPE 的概念及应用
  - 了解当前市场估值在历史中的位置，用估值框架判断长期回报区间
  - 记住：低估值期往往提供更好的长期机会

  3. 制定资产配置策略
  - 依据“100–年龄”设定股票比例，并结合收入稳定性与心理承受力微调

  4. 开设必要的投资账户
  - 开通证券账户用于 ETF；了解税收优惠账户（如 401k/IRA 等）的优势
  - 对比券商费率与体验，选择低费用、服务好的平台

  ### 30天内：启动股票投资实践

  1. 建立核心股票投资组合
  - 按配置比例购买选定指数基金
  - 优先建立美国股市敞口（标普500），再做国际分散化（发达/新兴市场）
  - 记录每次投资的时间、金额、价格与理由

  2. 设置定期投资计划
  - 每月定投 15–25% 月收入，固定日期，尽量自动化
  - 使用“美元成本平均法”降低择时风险

  3. 学习应对市场波动的心理准备
  - 学习行为金融学常见偏差（过度自信、损失厌恶等）
  - 市场下跌时保持冷静，将下跌视为更低价格买入的机会
  - 建立投资日记，记录波动期心理状态

  4. 建立投资知识体系
  - 学习企业基本面、宏观与行业分析
  - 关注权威教育内容，避免短期投机信息
  - 理解长期回报来源于盈利增长与股息

  ### 60-90天：完善长期投资体系

  1. 完成目标投资组合构建
  - 达到目标股票比例并确保分散化
  - 评估组合总费用率（目标 ≤0.5%）
  - 建立充足现金储备，避免被迫在不利时机卖出

  2. 建立再平衡机制
  - 偏离目标 5–10% 时再平衡；每年一次或大幅波动后执行
  - 再平衡是“卖高买低”的纪律保障

  3. 优化税收效率
  - 最大化利用税优账户；在应税账户优选税收效率高的指数基金
  - 尽量持有超过一年以享受长期资本利得税

  4. 制定长期投资规划与应急预案
  - 设定 10/20/30 年目标
  - 牛市不追高，熊市坚持定投甚至增持
  - 建立纪律检查清单，定期自我评估
